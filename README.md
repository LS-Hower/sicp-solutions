# sicp-solutions

我的 SICP 习题解答（[页面链接](https://ls-hower.github.io/sicp-solutions/)）

## 说明

SICP，全名为《计算机程序的构造和解释》（*Structure and Interpretation of Computer Programs*），作者为 Harold Abelson、Gerald Jay Sussman 和 Julie Sussman。

这里使用本书第二版。

书中使用 [Scheme 编程语言](https://www.scheme.org/) 。本仓库使用 [Racket 编程语言](https://racket-lang.org/) 。后者是前者的超集。

本仓库使用了：

- Racket 文档工具 [Scribble](https://docs.racket-lang.org/scribble/) ，用于生成网页，便于浏览。
- Racket 包 [SICP Collections](https://docs.racket-lang.org/sicp-manual/index.html) ，便于在代码中引用在书中定义的实用过程等对象。

仓库目录：

- `src/` ：习题解答的源代码
- `doc/` ：生成的 HTML 网页
